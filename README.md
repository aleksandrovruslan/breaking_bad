### Приложение с информацией о сериале Breaking Bad
#### В приложении используется:
* Чистая архитектура
* MVVM
* RxJava3
* Dagger2
* Room
* Okhttp3
* Glide
* Апи предоставлено сайтом https://breakingbadapi.com/
* Документация апи https://breakingbadapi.com/documentation
